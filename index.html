remoteagent by mheadd

A complete remote agent / remote call center solution built with CouchDB, Tropo and the Phono SDK.

RemoteAgent is a call center application that lives in the cloud.

It's a CouchApp that serves Phono instances directly from a CouchDB instance, and uses the built in features if CouchDB to deliver the SIP address of an available agent to a telephony engine.

RemoteAgent support advanced call center features like custom SIP headers and screen pops. It can be run using a combination of Asterisk and Node.js, or it may be run entirely in the cloud using the Tropo cloud communication platform.

Dependencies

Use of this solution requires:

  • A Tropo account.
  • A Phono API key.
  • An instance of the open source NoSQL database, CouchDB.
  • The CouchApp command line utility for deploying CouchApps.
